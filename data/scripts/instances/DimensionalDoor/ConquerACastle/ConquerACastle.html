Conquer a Castle:

General Information:
Min/Max Players: 7/7
Type: PvE
Instance time: 60 Minutes
Reuse: Every day at 6:30AM
Difficulty: Hard
Status: Available
Description:
This instance simulates a Castle siege vs the Elite clan (NPCs), there are many crystals in the castle that should be destroyed in order to be able to kill the special raid that resides inside the castle.


Reward Information: On this instance you will get Shiny Coin, the amount depends on the instance difficulty and instance minimum players.
These coins can be used at the Dimensional Door Shop.

Available Actions: Enter to the instance.
